Kubernetes Workshop: Intermediate

Introduction to CNCF projects with Kubernetes. This workshop is best suited for attendees who would like to know more about the power of Kubernetes beyond the default configurations. This workshop introduces attendees to several CNCF projects which compliments the Kubernetes cluster.

This workshop is structured for attendees with basic Kubernetes knowledge and some experience working with Kubernetes. Ideally, the attendees should have completed [Kubernetes Workshop: Fundamental]().

Agenda

This workshop introduces attendees to a few popular CNCF projects that compliments the operation of a Kubernetes cluster. The workshop guides the attendees through a practical scenario where a microservice architecture is required to be deployed in Kubernetes with good practices in place.

Scenario

You are an infrastructure developer tasked with deploying a BookInfo application in Kubernetes. This application is essentially a reviews and rating system for books. This application is composed of a suite of microservices written in multiple languages (Java, Javascipt, Python and Ruby). Besides successfully deploying theses services, you should also consider good practices including scalability, security and high availability.

Approach

Attendees will be introduced to 6 popular CNCF projects which are used to solve the above scenario. The projects selected for this workshop have mixed maturity levels. This exposes the attendees to production-grade technologies as well as cutting edge projects in order to spark innovation.

The CNCF projects that are introduced in this workshop:

  • Kubernetes Graduated
  • Prometheus Graduated
  • Envoy (Istio) Graduated
  • Jaegar Incubating
  • Helm Incubating
  • Telepresence Sandbox

Key Focus Areas

This workshop has 3 key focus areas:

  • Kubernetes toolset exploration
  • Cloud native development approaches
  • Cloud infrastructure good practices

Laboratories

The above key focus areas are further expanded to 7 laboratory sessions:

  1. Getting to know the Kubernetes environment
    • Overview of the Kubernetes architecture
    • Using Kubernetes-dashboard
  2. Deploying the BookInfo application
    • Overview of the microservice architecture
    • BookInfo as microservices
  3. Observing with Prometheus
  4. Tracing with Jaegar
  5. Deploying with Helm
  6. Networking with Istio
  7. Developing with Telepresence

Outcome

By the end of the workshop the attendees will be competent with provisioning applications/services using Helm charts. The attendees will be able to gain insights to the health of services with Prometheus. If a service is broken, the debugging process can be carried out with Jaegar and Telepresence. The attendees are introduced to Istio and see how a service mesh can be used in traffic management and monitoring.
